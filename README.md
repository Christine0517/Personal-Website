# Personal-Website

A personal portfolio website for Xilin Christine Huang, showcasing her background in marketing and finance.

## About

This is a static HTML/CSS/JS website featuring:
- Hero section with animated elements
- About section with photo and skills
- Professional experience timeline
- Projects showcase
- Skills and interests
- Contact information

## Running Locally

To view the site locally:

1. Ensure you have Python 3 installed
2. Run: `python3 -m http.server 8000`
3. Open http://localhost:8000 in your browser

## Technologies Used

- HTML5
- CSS3 (with custom properties and animations)
- JavaScript (for dynamic leaf animation)
- Google Fonts (Cormorant Garamond, DM Sans)
